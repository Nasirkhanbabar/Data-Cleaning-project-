
This repository contains basic data cleaning examples using Python.  
It focuses on handling missing values, data preprocessing, and basic visualization.

## Tools & Libraries
MS Excel – for viewing and basic manipulation of datasets

Python – main programming language used

Pandas – for data manipulation and cleaning

NumPy – for numerical operations

Seaborn – for statistical data visualization

Matplotlib – for general plotting and visualization

## Purpose 
This project is created for learning and practice in data analysis and data science. It is ideal for beginners who want to understand:

How to handle missing values in datasets

How to preprocess and clean data before analysis

How to visualize data using Python libraries
